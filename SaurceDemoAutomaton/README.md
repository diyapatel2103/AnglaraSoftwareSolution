# SauceDemo Selenium Automation (No TestNG)

## Setup Steps
1. Import project in Eclipse as Maven Project
2. Run `mvn clean install`
3. Run `LoginTest.java` as Java Application

## Tools Used
- Selenium WebDriver
- Java
- Maven
- Page Object Model (POM)

## Test Cases
- Valid Login
- Invalid Login
- Empty Username
- Empty Password

## Output
- test-report.html generated after execution
